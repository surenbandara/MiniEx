# Deployment Instructions

## Backend Deployment
1. Install packages:
   ```bash
   npm install
   ```
2. Deploy MongoDB in Docker:
   ```bash
   npm run mongodb-build
   ```
3. Compile TypeScript to JavaScript:
   ```bash
   npm run build
   ```
4. Run the backend server:
   ```bash
   npm run run
   ```

## Frontend Deployment
1. Install packages:
   ```bash
   npm install
   ```
2. Run the frontend server:
   ```bash
   npm start
   ```
```
